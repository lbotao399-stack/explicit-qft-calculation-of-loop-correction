# Explicit QFT Calculation of Loop correction

这个目录现在只保留两类内容：

- notation / theory input
- working flow / guidance

所有旧 calculation pages、原始计算归档、枚举渲染脚本与中间结果都已清掉。

## 目录说明

- `main  working flow and result.md`
  - 工作入口。只保留核心对象定义、标准工作流与目录分工。
- `working guidance.md`
  - 工作指导文件。固定后续重建 calculation pages 时应遵守的统一格式与步骤。
- `framework_shift_log.md`
  - 简短记录当前采用的框架切换与冻结约定。
- `theory/`
  - 按理论与工作方式分桶放记号、作用量、SUSY、current、selection rules 与 workflow master pages。

## 建议阅读顺序

1. 先看 `working guidance.md`
2. 再看 `main  working flow and result.md`
3. 然后按理论进入 `theory/`

## 备注

- 后续若重建 calculation pages，统一以当前 `working guidance.md` 与对应 theory bucket 为源。
- 当前主线仍采用 `susy current diagram / anomaly Ward identity / local-term extraction` 口径。
